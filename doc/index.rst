Welcome to the documentation of the signac project template!
============================================================

This is the documentation for the **signac-project-template** designed for rapid project development based on the data management framework signac_ and the workflow extension signac-flow_.

.. note::

   Before reading this manual you should be familiar with the basic concepts of signac_.

.. _signac-flow: https://signac-flow.readthedocs.io/
